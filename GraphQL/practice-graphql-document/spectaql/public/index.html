GraphQLドキュメント練習用アプリケーション

GraphQLのドキュメント生成の練習用のアプリケーション。 下記の関連を持つ。

  • Product -> Member
  • Product -> Service
API Endpoints
http://localhost:8080/graphql
Version

1.0.0

Queries

findAllProducts

Description

全てのProductを取得する

Response

Returns a Products!

Example

Query
query findAllProducts {
  findAllProducts {
    products {
      id
      code
      name
      members {
        id
        givenName
        familyName
        birthDay
      }
      services {
        id
        name
        kind
      }
    }
  }
}
Response
{"data": {"findAllProducts": {"products": [Product]}}}
Queries

findProduct

Description

パラメータidに指定されたProductを取得する。 Productが存在しない場合はエラーになる。

Response

Returns a Product!

Arguments
Name Description
id - String! 取得したいProductのIDを指定する。

Example

Query
query findProduct($id: String!) {
  findProduct(id: $id) {
    id
    code
    name
    members {
      id
      givenName
      familyName
      birthDay
    }
    services {
      id
      name
      kind
    }
  }
}
Variables
{"id": "abc123"}
Response
{
  "data": {
    "findProduct": {
      "id": "xyz789",
      "code": "xyz789",
      "name": "xyz789",
      "members": [Member],
      "services": [Service]
    }
  }
}

Mutations

registerProduct

Description

新しくProductを登録する。

Response

Returns a RegisterProductResult!

Arguments
Name Description
input - RegisterProductInput! 登録したいProductの情報を入力する。

Example

Query
mutation registerProduct($input: RegisterProductInput!) {
  registerProduct(input: $input) {
    id
    product {
      id
      code
      name
      members {
        id
        givenName
        familyName
        birthDay
      }
      services {
        id
        name
        kind
      }
    }
  }
}
Variables
{"input": RegisterProductInput}
Response
{
  "data": {
    "registerProduct": {
      "id": "abc123",
      "product": Product
    }
  }
}
Mutations

updateProduct

Description

Productの情報を更新する。 更新対象が存在しない場合はエラー。

Response

Returns an UpdateProductResult!

Arguments
Name Description
input - UpdateProductInput! 更新したいProductの情報を入力する。

Example

Query
mutation updateProduct($input: UpdateProductInput!) {
  updateProduct(input: $input) {
    id
    product {
      id
      code
      name
      members {
        id
        givenName
        familyName
        birthDay
      }
      services {
        id
        name
        kind
      }
    }
  }
}
Variables
{"input": UpdateProductInput}
Response
{
  "data": {
    "updateProduct": {
      "id": "xyz789",
      "product": Product
    }
  }
}

Types

Member

Description

Memberの情報。

Fields
Field Name Description
id - String! Memberをユニークに特定するためのID。
givenName - String! Memberの名前。
familyName - String! Memberの苗字。
birthDay - Date! メンバーの誕生日。

  "familyName": "xyz789",
  "birthDay": "2007-12-03"

Product

Description

Productの情報。

Fields
Field Name Description
id - String! Productをユニークに特定するためのID。
code - String! Productのコード。
name - String! Productの名前。
members - [Member!]! Productに携わるMemberのリスト。
services - [Service!]! Productに含まれるServiceのリスト。

  "name": "abc123",
  "members": [Member],
  "services": [Service]
